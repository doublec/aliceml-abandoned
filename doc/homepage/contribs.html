This page is intended as a repository for Alice-related libraries, tools, and other contributions by the user community. Please post additions, updates, corrections and report broken links to

  • stockhausen@ps.uni-sb.de
  • AliceParsec by Benedikt Grundmann.
    Port of Daan Leijen's Parsec parsing combinator library for Haskell.

  • Functional Graph Library by Guido Tack.
    Port of Martin Erwig's FGL, which is based on inductive graphs.

  • Purely Functional Data Structures by Guido Tack.
    Port of the data structures from Chris Okasaki's book, including lazy data structures.

  • Functional XML Parser by Guido Tack.
    Port of Andreas's Neumann's fxp.

  • Regular Expression Library by Guido Tack.
    Port of the SML/NJ RegExp library.

Libraries in this section can only be used with Alice versions from 1.0 on.

  • GD Graphics Library by Guido Tack.
    A binding for the GD library that provides functions for creating and manipulating JPEG and PNG images, plus a high-level declarative interface.

  • SQLite by Guido Tack.
    A binding for SQLite. Gives Alice access to SQL databases.

Libraries in this section can only be used with Alice versions up to 0.9.3.

  • Graph Data Structure by Sebastian Miele.
    Binding to the Graph Data Structure for Mozart, which provides imperative graphs.
